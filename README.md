# MPV 极速定制优化配置 - Intel UHD 630 绝杀版

这是一个深度为 **Intel UHD 630 核显**（如 i7-8750H, i5-8400 等老牌强芯）量身定制的 `mpv` 播放器配置文件项目。

它的核心精神是：**“彻底压榨核显性能，强制独显待机静音，斩断一切流媒体断流与卡顿的可能。”**

---

## 🎯 核心优化成果

### 1. 极致低发热：真正的 100% 旁路独显
抛弃低效软解与费电的独显（如 GTX 1060）。通过锁死 `d3d11-adapter=Intel` 和 `hwdec=d3d11va` (零拷贝机制)，强迫所有的解码、排版、缩放流水线在 UHD 630 内部通过显存闭环完成。
* **表现：** 双开 1080P 直播，CPU 利用率低于 15%，独显深度休眠，**笔记本风扇彻底安静。**

### 2. 八风不动的网络流底盘
在看 IPTV 和 `m3u8` 源时再也不怕网络抖动和服务器封禁：
* 注入 FFmpeg 原生通讯通道 `demuxer-lavf-o=reconnect=1`，网络闪断直接后台无感重连。
* `400MiB` 超大预卷蓄水池 + `cache-pause=yes` 机制，彻底消除“卡一下喘一下”的恶劣体验。

### 3. YouTube / B站 聪明解析 (抗闪退 & 防 AV1 卡死)
借助精心编写的 `yt-dlp` 解析正则表达模块：
* 强力避雷由于老旧核显不支持 **AV1 编码** 而引发的 CPU 百分百负载和闪退崩溃。
* **自适应保底策略：** 优先索要最高画质非 AV1 的流（比如 VP9，H.265），即使视频是纯 AV1 也会进行智能降级播放，达到 **100% 的网络链接拖拽解析率**。

### 4. 解决千古顽疾：老驱动下的各种黑屏与闪烁
针对 2020 年代的传统核显驱动完美避坑：
* 退守经典拷贝 Flip 模式 或 关闭 `vd-lavc-dr`，彻底根治缩放黑屏与“暂停视频切后台后画面永久丢失” 的惨案。
* 色彩模式退回 `auto` / `rgb10_a2` 修复 HDR 假信号造成的“满屏通红” 和“刺眼蓝光”。

---

## 📁 配置文件指南

本项目提供了两种不同场景的配置文件。使用时，请挑选适合你的文件，将其重命名为 **`mpv.conf`** 并放置在 `portable_config` 文件夹下。

### 💻 `mpv.conf` (笔记本 / 1080P 移动神油版)
* **适用对象：** i7-8750H / 1080P 屏幕。
* **特点：** 启动自动最小化压后台。将网络流的最高解析画质坚决锁死在 `1080P`。完美适应分屏双开两路直播的生产力需求。

### 🖥️ `mpv-4k.conf` (桌面主机 / 4K 巨屏满血版)
* **适用对象：** i5-8400 / UHD 630 桌面主机 / 4K 显示器用户。
* **特点：** 解除 1080P 解析封印。强制逼迫流媒体服务器推送 UHD 630 能完美核芯硬解的 **4K VP9** 高画质源。同时开启了 DPI 高分屏缩放适配，并将发车基础窗口扩展成 1080P，带来极致的巨屏视听盛宴。

---

## 🛠️ 食用方法 & 准备工作

1. 请确保你使用的是官方较新的 `mpv` Windows 版本。
2. 确保在 `mpv.exe` 同一目录下放置了最新版本的 **`yt-dlp.exe`**。（这对于实现在线拖拽播放必不可少）。
3. 复制你选定的 `.conf` 内容到 `portable_config` 中。
4. **尽情享受极致丝滑的多媒体体验吧！**
